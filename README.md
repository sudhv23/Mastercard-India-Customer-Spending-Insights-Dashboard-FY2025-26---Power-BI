# 📊 Mastercard India – Customer Spending Insights Dashboard (FY2025–26)

## 🧾 Overview
This project showcases a **Power BI dashboard** analyzing customer spending behavior for Mastercard in India during **FY2025–26 **.

The dashboard is designed from a **CFO/Business perspective**, focusing on key financial and operational metrics such as spend trends, transaction patterns, customer activity, and market positioning within India’s credit card ecosystem.

---

## 🎯 Objectives
- Analyze **monthly spending trends** aligned to financial year  
- Identify **top spending categories and cities**  
- Understand **transaction distribution across spend buckets**  
- Evaluate **channel usage (Online vs POS)**  
- Provide **market context through network share analysis**

---

## 📊 Key Insights
- **Total Spend:** ₹23.7 Cr  
- **Top Category:** Travel (₹6.8 Cr)  
- **Online Spend Share:** 75.2%  
- **Top City:** Mumbai (₹4.6 Cr)  
- **Transaction Concentration:** ₹1K–₹5K range  
- **Peak Period:** Oct–Nov (festive season)  
- **Market Share:** ~22% (Mastercard in India)

## 🧠 Data Model & Calculations
- Designed using a **star schema model**
- Implemented **DAX measures** for KPIs and aggregations
- Applied **financial year logic (April–March)** using Calendar table
- Created **custom sort logic** for spend buckets
- Standardized metrics in **₹ Crores (Cr)** for business relevance
